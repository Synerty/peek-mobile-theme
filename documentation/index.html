Peek App Documentation

Main Navigation

The top navigation is not designed with conventional Bootstrap elements as it is designed to only expand the title section while the buttons on the left and the right remain a fixed size throught a screen resize. Apply the following code sample to get the main navigation to display on your page.

Display:

Peek Title
Code sample:
<peek-main-title _nghost-vob-4=""> <div _ngcontent-vob-4="" class="container-fluid"> <div _ngcontent-vob-4="" class="menu-bar"> <div _ngcontent-vob-4="" class="left"> <button _ngcontent-vob-4="" class="btn left-button " ng-reflect-router-link="/">Home</button> </div> <div _ngcontent-vob-4="" class="menu-title"> <h2 _ngcontent-vob-4="">Peek Title</h2> </div> <div _ngcontent-vob-4="" class="right"> <div class="btn-group" role="group" aria-label=""> <button _ngcontent-vob-4="" class="btn right-button" ng-reflect-router-link="/peek_plugin_active_task">Tasks</button> <button _ngcontent-vob-4="" class="btn right-button" ng-reflect-router-link="/peek_plugin_active_task">User</button> </div> </div> </div> </div> </peek-main-title>

Footer

Use the following code just before the closing of </peek-main-app> sample to display the footer. Display:

Offline, xxx minutes
Code sample:
<peek-main-footer> <div class="container-fluid"> <div _ngcontent-vob-5="" class="row"> <div _ngcontent-vob-5="" class="col-lg-12 col-md-12 col-sm-12 col-xs-12 text-center"> <h2> Offline, xxx minutes </h2> </div> </div> </div> </peek-main-footer>

Home Page

Background

The home page background will look different to the backgrounds on subsiquent pages. The main idea of having the home page look different is it make it look a bit more impressiv and welcoming. Apply the class 'home-page' to the page body if you would like this background to display.

Display:
Code sample:
<body class="home-page">...</body>
