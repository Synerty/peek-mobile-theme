Peek App Documentation

Main Navigation

The top navigation is not designed with conventional Bootstrap elements as it is designed to only expand the title section while the buttons on the left and the right remain a fixed size throught a screen resize. Apply the following code sample to get the main navigation to display on your page.

Display sample:

Peek Home
Code sample:
<peek-main-title> <div class="container-fluid"> <div class="menu-bar"> <div class="menu-bar-left"> <button class="btn left-button">Home</button> </div> <div class="menu-title"> <h2>Peek Title</h2> </div> <div class="menu-bar-right"> <div class="btn-group" role="group" aria-label=""> <button class="btn right-button">Tasks</button> <button class="btn right-button">User</button> </div> </div> </div> </div> </peek-main-title>

Pages

Home Page

Background

The home page background will look different to the backgrounds on subsiquent pages. The main idea of having the home page look different is it make it look a bit more impressiv and welcoming. Apply the class 'home-page' to the page body if you would like this background to display.

Display:
Code sample:
<body class="home-page">...</body>

Buttons

Buttons that represent apps on the home page are not using fontawesome. They are a combination of an image and text.

Display Sample:

Testing Noop
Code sample:
<a href=""> <img src="../images/testing_noop.png"> <p class="text-center">Testing Noop</p> </a>
